# lissasfa-top

Ce dépôt inclut maintenant une application Next.js minimale pour une plateforme de cours.

## Développement local

Exécuter localement :

```bash
cd /workspaces/lissasfa-top
npm install
npm run dev
```

Identifiants administrateur par défaut : `admin` / `admin`.

Les données vidéo sont stockées dans `videos.json` et les utilisateurs dans `users.json`.
